# ProConsulta

A complete management system for doctors and pacients schedules.

## Technologies and Practices Used

- **Blazor with .NET 8**
- **MUDBlazor**
- **Entity Framework Core**
- **Render Modes**
- **Components**
- **Forms**
- **Forms Validations**
- **Authentication and Authorization with Identity**
- **Database Integration with SQLServer**
